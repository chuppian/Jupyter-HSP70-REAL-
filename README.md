# HSP70 Comparative Bioinformatics Portfolio

## Overview
This project investigates the conservation of the HSP70 protein family across human, mouse, Arabidopsis and E. coli.

## Biological Question
How are members of the HSP70 protein family conserved across different organisms and what structural and functional differences exist between them?

## Data Sources
- UniProt REST API
- RCSB PDB API

## Files
- HSP70.ipynb
- HSP70.fasta 
  
## Requirements
- Python 3.x
- requests
- pandas

## Running the Notebook
1. Clone the repository
2. Open the notebook in Jupyter or VS Code
3. Run all cells in order
